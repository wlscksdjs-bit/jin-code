# 원가 등록 고도화 작업지시서 (v1.0)

**작성일**: 2026-03-22  
**프로젝트**: pms-new  
**대상 기능**: 실행원가 등록 (`/cost/[projectId]/execution`)  
**목표**: 직접비/간접비 구분 + 다중업체 지원 + Excel 일괄 등록

---

## 1. 현재 상태 분석

### 1.1 현재 실행원가 등록 UI 문제점

| # | 문제 | 현재 구현 | 개선 필요 |
|---|------|----------|----------|
| 1 | **직접비/간접비 구분 없음** | 19개 항목 평면 나열 | 그룹별 분류 필요 |
| 2 | **외주비 다중업체 미지원** | `outsourceFabrication: number` | 업체별 상세 내역 필요 |
| 3 | **엑셀 업로드 없음** | 수동 숫자 입력만 | Excel 파일 업로드 필요 |
| 4 | **엑셀 템플릿 없음** | 없음 | 양식 제공 필요 |
| 5 | **일괄 등록 없음** | 월별 1개 등록 | 다수 월 동시 등록 |

### 1.2 현재 데이터 모델

```prisma
model CostExecution {
  // ... 기존 필드
  outsourceFabrication    Float  // 외주비(제작) - 단일 합계
  outsourceService       Float  // 외주비(용역) - 단일 합계
  // ...
  items                  CostExecutionItem[]  // 상세 항목 (vendor 필드 있음)
}

model CostExecutionItem {
  vendor        String?   // 업체명 (현재 미활용)
  itemName      String   // 항목명
  amount        Float    // 금액
  // ...
}
```

### Task 8: 원가 요약 컴포넌트

**파일**: `src/components/cost/CostSummary.tsx`

**표시 항목**:
| 항목 | 계산식 |
|------|--------|
| 계약금액 | 입력값 |
| 직접비 합계 | 재료비 + 노무비 + 외주제작 + 외주용역 |
| 간접비 합계 | 나머지 항목 합계 |
| 총製造원가 | 직접비 + 간접비 |
| 판관비 | 총원가 × 판관비율 |
| 총원가 | 제조원가 + 판관비 |
| 매출이익 | 계약금액 - 총원가 |
| 이익률 | (매출이익 / 계약금액) × 100 |
