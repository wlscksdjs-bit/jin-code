## 핵심 데이터 흐름 (현재 상태 → 목표)

### 현재
```
프로젝트 → 견적원가 → 예산 → 실행원가 → 월별실적
              ↓
         (고립된 모듈)
```

### 목표
```
입찰관리 → 견적원가 → 수주확정 → 예산배정 → 실행원가 → 월별실적 → 손익예측
                              ↓                              ↓
                         설계/발주/시공 일정              자금수지 실시간
                              ↓                              ↓
                         [통합 간트차트] ←────────────── [실시간 대시보드]
```

## KPI 정의

| KPI | 계산식 | 의미 |
|-----|--------|------|
| **CPI** | EV / AC | 원가성과지수 (>1: 양호, <0.9: 위험) |
| **SPI** | EV / PV | 일정성과지수 (>1: 양호, <0.9: 위험) |
| **EAC** | BAC / CPI | Complete 예상비용 |
| **VAC** | BAC - EAC | 최종 variance (+: 절감, -: 초과) |
