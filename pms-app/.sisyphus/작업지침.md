# PMS-App 전면 재설계 작업 지침

## 개요

**프로젝트명**: pms-app 전면 재설계
**핵심**: 원가 관리 중심 프로젝트 관리 시스템
**방식**: 전체 신규 작성 (점진적 확장 아님)

---

## Tech Stack

| 항목 | 선택 |
|------|------|
| Framework | Next.js 16+ |
| ORM | Prisma 5+ |
| Database | PostgreSQL |
| Realtime | SSE (Server-Sent Events) |
| Auth | NextAuth 5 |
| Validation | Zod |
| UI | Radix UI + Tailwind CSS 4 |
| Gantt | 외부 라이브러리 (dhtmlxGantt 또는 Pragmatic DnD) |
| Testing | vitest + Playwright |

---

## 데이터 흐름 (핵심)

```
입찰관리 → 견적원가 → 수주확정 → 예산배정 → 실행원가 → 월별실적 → 손익예측
                              ↓
                         설계/발주/시공 일정
                              ↓
                   자금수지 실시간 + 대시보드
```

---

## Phase별 작업 내용

### Phase 1: Foundation (Tasks 1-6)
- 프로젝트 스캐폴딩 + PostgreSQL 설정
- Prisma Schema 설계
- NextAuth 인증
- RBAC 권한
- UI 컴포넌트 기본
- 레이아웃 + 네비게이션

### Phase 2: 원가 핵심 (Tasks 7-12)
- 프로젝트 CRUD
- 견적원가 관리 (WBS 기반)
- 예산 관리 (견적 → 예산 전환)
- 실행원가 모델

### Phase 3: 실행원가 + 실적 (Tasks 13-17)
- 실행원가 UI
- 월별 실적 시스템
- SSE 실시간 Push
- Dashboard 집계 테이블

### Phase 4: 일정/구매 (Tasks 18-22)
- WBS/CPM 일정관리
- 간트차트 연동
- 발주/입고 관리

### Phase 5: 재무/대시보드 (Tasks 23-27)
- 자금수지 실시간
- KPI 카드 (CPI/SPI/EAC/VAC)
- 수주영업 현황
- 대시보드 통합

---

## KPI 정의

| KPI | 계산식 | 의미 |
|-----|--------|------|
| **CPI** | EV / AC | 원가성과지수 (>1: 양호) |
| **SPI** | EV / PV | 일정성과지수 (>1: 양호) |
| **EAC** | BAC / CPI | Complete 예상비용 |
| **VAC** | BAC - EAC | 최종 variance |
